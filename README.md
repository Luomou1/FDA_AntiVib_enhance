# FDA_Antivib

`FDA_Antivib` 是一个面向白光扫描干涉数据分析的桌面应用，当前支持 `FDA` 与 `PhaseGap` 两条处理流程，并集成了有效扫描范围自动检测能力。项目从原有 `Phase_Gap` 工作基础上拆分出来，目标是把抗振场景下的干涉分析流程收敛为一个可直接运行、可视化、便于后续维护的独立仓库。

## 功能概览

- 提供基于 `PySide6` 的桌面 GUI，便于交互式选择数据与参数
- 支持 `FDA` 与 `PhaseGap` 两种分析模式切换
- 自动检测 active scanning range，并支持在检测结果基础上做左右扩展
- 支持自动估计全局 `K0`，也支持手动指定固定 `K0`
- 提供频谱长度档位选择，兼顾分析精度与运算成本
- 输出高度、相位、拟合斜率等关键结果，并配套图形查看

## 处理流程

1. 读取图像序列或体数据
2. 沿扫描维度计算干涉调制强度，自动确认 active scanning range
3. 自动估计全局 `K0`，或读取用户指定的固定 `K0`
4. 对逐像素信号去直流
5. 施加窗函数并执行频谱分析
6. 在全局 `K0` 附近截取局部拟合窗口
7. 对局部相位解包裹并拟合得到 `g0` 与 `phi0`
8. 计算基础高度 `h = -g0 / 2`
9. `FDA` 模式下执行整数级次修正，得到 `h_prime`
10. `PhaseGap` 模式下在 FDA baseline 结果上继续做 phase-gap connect / fit / finalization，输出最终 `h_prime`

## 主要输出

- `h`：由频域相位斜率得到的基础高度图
- `h_prime`：最终修正后的高度图
- `phi0`：全局 `K0` 处的相位图
- `g0`：局部相位拟合斜率
- `active_range`：自动识别出的有效扫描帧范围

## 频谱长度策略

界面支持以下频谱长度档位：

- `1x`：FFT 长度等于有效帧数
- `2幂次第1档`：第 1 个严格大于有效帧数的 2 的幂
- `2幂次第2档`：第 2 个严格大于有效帧数的 2 的幂
- `2幂次第3档`：第 3 个严格大于有效帧数的 2 的幂
- `2倍 / 4倍 / 8倍补0`：直接按有效帧数倍增后补零

如果启用“扩大有效范围”，程序会在自动检测到的范围左右两侧按设定帧数扩展，并自动裁剪到数据边界内。扩展后的范围会同时参与自动 `K0` 估计和正式分析。

## 项目结构

```text
FDA_Antivib/
├─ main.py                  # 桌面应用入口
├─ pyproject.toml           # 项目元数据与依赖声明
└─ app/
   ├─ core/                 # FDA / PhaseGap 核心分析逻辑
   ├─ gui/                  # 主界面、线程与交互窗口
   ├─ pipeline/             # 数据读取、session、active range 检测
   └─ plotting/             # 绘图与可视化辅助
```

## 环境要求

- Python `3.11+`
- 推荐在 Windows 环境下运行桌面界面

## 安装方式

先创建虚拟环境，再安装项目依赖：

```powershell
python -m venv .venv
.venv\Scripts\Activate.ps1
pip install -U pip
pip install -e .
```

如需安装测试依赖：

```powershell
pip install -e .[test]
```

## 运行方式

在项目根目录执行：

```powershell
python main.py
```
